# SM Living Website

Sito web ufficiale per SM Living - Sviluppo immobiliare di qualità in Brianza.

## Progetti

- **Residenza Excelsior Living** - 24 unità abitative ad Agrate Brianza
- **Residenza Lilium Living** - 6 unità esclusive a Bellusco

## Tecnologie Utilizzate

- React 18
- React Router per la navigazione
- Vite come build tool
- CSS moderno e responsive

## Installazione Locale

```bash
# Installa le dipendenze
npm install

# Avvia il server di sviluppo
npm run dev

# Build per produzione
npm run build
```

## Deployment su Vercel

Questo progetto è configurato per il deployment automatico su Vercel:

1. Collega il repository GitHub a Vercel
2. Vercel rileverà automaticamente le configurazioni
3. Ogni push su main attiverà un deployment automatico
